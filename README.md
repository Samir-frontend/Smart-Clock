## 🕐 Overview

> A **Smart Clock** with concentric rotating rings showing live date, month, and day — combined with a smooth analog clock face. Built purely with HTML5 Canvas and vanilla JavaScript.

---

## ✨ Features

- 🔴 **Outer Ring** — Date (01–31), current date highlighted in **red**
- 🔵 **Middle Ring** — Month (JAN–DEC), current month highlighted in **blue**
- 🟢 **Inner Ring** — Day (SUN–SAT), current day highlighted in **green**
- ⚪ **Analog Clock** — Smooth hour, minute & second hands with glow effect
- 🔴 **Red second hand** with center dot accent
- ⏱️ **Digital time + full date** displayed below
- 📱 **Fully responsive** — auto-scales to any screen size
- 🎯 **Rings rotate smoothly** — current item always at top (12 o'clock)

---

## 🎨 Preview

> Dark theme with glowing highlights — rings rotate in real-time as time, date, and month change.

---

## ⚙️ How It Works

```javascript
1. Canvas API renders all 3 concentric rings every frame
2. Each ring rotates so current item stays at 12 o'clock (top)
3. Smooth lerp animation for ring rotation
4. requestAnimationFrame() keeps everything silky smooth
5. Analog hands update with millisecond precision
```
